# 📌 Project Overview

The **Customer Shopping Behavior Analysis Dashboard** is an end-to-end data analytics project developed to analyze customer purchasing patterns, spending behavior, product preferences, and business performance.

Using **Python, SQL, Streamlit, and Power BI**, this project converts raw customer transaction data into meaningful insights that support data-driven business decisions.

The dashboard enables businesses to understand customer behavior, identify high-value customers, optimize sales strategies, and improve overall profitability.

---

# 🎯 Project Goal

The primary goal of this project is to:

> Analyze customer shopping behavior and provide actionable business insights that help organizations improve customer satisfaction, increase sales, and enhance decision-making.

# 💼 Business Problem

Retail businesses collect large amounts of customer data but often struggle to convert it into actionable insights.

This project addresses key business questions:

- Who are the most valuable customers?
- Which products perform best?
- How does customer behavior affect sales?
- What factors influence purchasing decisions?
- Which customer segments contribute the most revenue?
- How can businesses improve retention and profitability?
